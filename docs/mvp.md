# DopamineWar MVP

## Goal

Android app blocks selected apps. Trusted people approve temporary access through Telegram.

## Scope

- Android app with minimal UI
- Main UX through Telegram bot
- No external server
- Telegram Bot API via polling on the phone
- Block selected apps, initially YouTube and Instagram
- Access durations: 5, 10, 15 minutes, or custom up to 80 minutes
- Approval wait timeout: 10 minutes
- Trusted people registration through invite link
- Bot token is entered manually in the app UI and is not stored in APK
